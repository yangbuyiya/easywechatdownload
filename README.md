## 功能概览

| 模块 | 功能 |
|------|------|
| 单篇下载 | 粘贴文章链接，一键保存为 HTML / Markdown / PDF |
| 批量下载 | 通过本地代理捕获公众号历史文章列表，批量归档 |
| 监控下载 | 开启监控后在微信中逐篇打开文章，自动捕获并下载 |
| EPUB 生成 | 将已下载的文章（HTML / Markdown）转换为电子书 |
| 资源本地化 | 自动下载文章中的图片和音频到本地 |
| 评论保存 | 抓取文章精选评论及回复，支持独立存储 |
| MySQL 同步 | 下载内容同步写入 MySQL 数据库 |
| 自动更新 | 支持 GitHub Release 自动检测和安装更新 |

## 使用指南

### 单篇文章下载

1. 复制微信公众号文章链接
2. 粘贴到首页输入框
3. 点击「下载文章」

### 批量下载

1. 前往系统设置安装并信任代理证书（首次使用）
2. 确保电脑版微信已登录
3. 点击「批量下载」启动本地代理
4. 在微信中打开目标公众号主页并刷新
5. 进入文章列表，代理自动捕获文章链接
6. 返回软件确认开始下载

### 监控下载

1. 点击「开始监控」按钮（按钮变为警告色表示已启动）
2. 在微信中逐篇打开需要下载的文章
3. 回到软件点击「停止监控」，开始下载已捕获的文章

### EPUB 电子书生成

1. 在侧边栏进入「创建电子书」
2. 填写电子书名称
3. 选择数据来源文件夹（包含已下载的 HTML/Markdown 文件）
4. 可选：添加封面图片
5. 点击「开始生成」

### 系统设置

| 配置项 | 说明 |
|--------|------|
| 保存路径 | 文章下载的本地存储目录 |
| 缓存路径 | 临时文件存放目录 |
| 导出格式 | HTML / Markdown / PDF（可多选） |
| 资源选项 | 下载图片、音频、评论等 |
| 下载间隔 | 批量下载时的间隔时间（毫秒） |
| 批量数量 | 单批处理的文章数量上限 |
| MySQL | 数据库连接配置，需先执行 `doc/mysql.sql` 建表 |
